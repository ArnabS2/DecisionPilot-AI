# DecisionPilot AI

DecisionPilot AI is an AI-powered decision support system that helps users make smarter decisions before taking important actions.

## Features

- AI-based decision analysis
- Option comparison
- Confidence score
- Risk analysis
- Regret prediction
- Future outcome simulation
- Decision history
- PDF report generation
- Mobile-friendly Streamlit UI

## Tech Stack

- Python
- Streamlit
- Gemini API
- Pandas
- Plotly
- ReportLab

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
